## Project Overview

- HTML Reports (HTML帳票)をReactコンポーネントとして作成・管理するリポジトリ
- ブラウザで表示し、PDF出力することを前提としている

## Tech Stack

- React 19 + TypeScript + Vite
- paper-css
- CSS

## Commands

- `npm install` — 依存パッケージのインストール
- `npm run dev` — 開発サーバー起動
- `npm run build` — ビルド（tsc + vite build）
- `npm run lint` — ESLint実行
