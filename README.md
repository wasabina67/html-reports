# html-reports
HTML Reports

## Setup

This project was scaffolded with [Vite](https://vite.dev/) using the `react-ts` template:

```bash
npm create vite@latest . -- --template react-ts
```

## Dev

```bash
npm install
```

```bash
npm run dev
```

http://localhost:5173/html-reports/

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```
